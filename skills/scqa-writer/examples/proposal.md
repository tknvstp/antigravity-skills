# 提案开场模板

## 适用场景
- 向领导提议新方案
- 申请资源或预算
- 推荐新工具或流程

## 关键特点
- S 强调现有成绩，建立可信度
- C 突出瓶颈或风险，制造紧迫感
- Q 聚焦「如何解决」
- A 给出明确方案和预期收益

## 示例

### 情境
过去一年，我们团队的内容产出量提升了 50%，保质保量地完成了所有指标。

### 冲突
随着业务扩张，创作效率已达到瓶颈，人工润色的时间成本极高，导致响应速度变慢。

### 问题
如何在不增加人力成本的情况下提升创作效率

### 回答
建议引入 Claude AI 写作技能包，它能自动化处理 80% 的初稿和润色工作。

## 注意事项
- S 不要过长，1-2 句即可
- C 要有数据支撑（如「增加 40%」「延迟 3 天」）
- A 要有量化的预期收益
